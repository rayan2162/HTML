<body> - represents the contains of an HTML document


content Sectioning

<address> - provides contract info for person, org

<aside> - not related to the main doc, presented as sidebars or callout box

<footer> - contains info about the author of the section,copyright data

<header> - represents intro content, navigation aids

<h1> to <h6> - represents six level of section headings

<hgroup> - represents a heading grouped with any secondary content such as subheading

<div> - The generic container for flow content.Block level element.

<figcaption> - Represents a caption or legend describing the rest of the contents of its parent <figure> element.

<hr> - Represents a thematic break between paragraph-level elements

<li> - Represents an item in a list.

<ul> - unoreder list

<ol> - Represents an ordered list of items

<menu> - A semantic alternative to <ul>, but treated by browsers as no different than <ul>.

<p> - paragraph

<pre> - The text is typically rendered using a non-proportional, or monospaced, font.
